# How to Install - Step by Step Guide

1. Fork the repository 

2. Clone using the following repo link
   #### `git clone https://github.com/Adityalad-25/pccoe-acm-2023.git`

3. Go to the project folder and run npm install
   #### `cd pccoe-acm-2023`
   #### `npm install`

4. Now run your app
   #### `npm start`
